# Фронтенд системы мониторинга ИТ-угроз

## Описание проекта
Фронтенд-часть веб-приложения для сбора и обработки информации об ИТ-угрозах в корпоративной среде. Предоставляет пользовательский интерфейс для работы с системой.

## Структура проекта

### Основные компоненты

#### Стили
* `style.css` — основной файл стилей для всех шаблонов

#### Шаблоны страниц
* `login.html` — страница авторизации
* `employee_index.html` — страница создания заявки для сотрудников
* `employee_requests.html` — список заявок сотрудника
* `specialist_index.html` — панель специалиста с полным списком заявок
* `request.html` — детальная страница заявки
* `threat.html` — страница описания угрозы
* `error.html` — страница обработки ошибок

### Вспомогательные файлы
* `pre-commit-config.yaml` — конфигурация pre-commit хуков
* `check_filenames.py` — скрипт проверки названий файлов

## Установка

1. Клонирование репозиториев:
```bash
# Фронтенд
git clone https://github.com/max0194/threat-monitoring-frontend.git

# Бэкенд (требуется для работы)
git clone https://github.com/max0194/threat-monitoring-backend.git
```

2. Запуск бэкенд-сервисов:
```bash
# Перейдите в директорию бэкенда
cd threat-monitoring-backend/compose

# Запустите контейнеры
docker compose up -d

# Запустите основное приложение
cd ../
go run ./cmd/threat-monitoring/main.go
```

3. Доступ к системе:
* Основной интерфейс: `http://localhost:8080`
